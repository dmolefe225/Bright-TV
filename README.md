# Bright-TV

*Project Description*

This project focuses on analyzing user behavior and viewership patterns for BrightTV, a digital streaming platform aiming to increase its subscription base within the current financial year. The primary objective of the project is to generate meaningful insights that will support the Customer Value Management (CVM) team in making data-driven decisions to drive growth, improve user engagement, and optimize content strategy.

The analysis is based on a dataset containing detailed information about user profiles and viewer transactions. Each record represents an individual viewing session, providing granular insight into how subscribers interact with the platform. Key aspects of the dataset include user demographics, viewing times, frequency of usage, and content consumption patterns. Since the data is recorded in UTC, it is first converted to South African time to ensure accuracy and relevance in trend analysis.

Through this project, various dimensions of user engagement are explored, including peak viewing times, daily and weekly consumption trends, and differences in behavior across user segments. The project also examines the factors that influence content consumption, such as time of day, type of content, and user preferences. By identifying patterns in high and low engagement periods, the analysis aims to uncover opportunities for targeted interventions.

In addition, the project provides strategic recommendations on the type of content that can be introduced or promoted during periods of low viewership to boost engagement. It also outlines potential initiatives to attract new users and retain existing subscribers, such as personalized content recommendations, promotional campaigns, and improved user experience.

Overall, this project combines data analysis, business understanding, and strategic thinking to deliver actionable insights that can help BrightTV grow its user base, increase platform usage, and strengthen its competitive position in the streaming industry.
